## flutter_vignettes là gì?

A collection of fun Flutter experiments, created by gskinner, in partnership with Google. Nó tập trung vào việc cải thiện quy trình phát triển và chất lượng mã Flutter. Dự án nằm tại [gskinnerTeam/flutter_vignettes](https://github.com/gskinnerTeam/flutter_vignettes) và được duy trì bởi `gskinnerTeam`.

## Cài đặt flutter_vignettes

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  flutter_vignettes: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_vignettes/flutter_vignettes.dart';
```

Xem thư mục `example/` và [trang pub.dev](https://pub.dev/packages/flutter_vignettes) để biết API chính xác — flutter_vignettes được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
