## mcp_flutter là gì?

MCP Toolkit for Flutter AI Agent Driven Development (MCP/CLI + custom client side tools) - via closed feedback loop (visual & semantic snapshot) and high client. Nó tập trung vào việc đưa AI (trên thiết bị hoặc đám mây) vào ứng dụng Flutter. Dự án nằm tại [Arenukvern/mcp_flutter](https://github.com/Arenukvern/mcp_flutter) và được duy trì bởi `Arenukvern`.

## Cài đặt mcp_flutter

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  mcp_flutter: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```

Xem thư mục `example/` và [trang pub.dev](https://pub.dev/packages/mcp_flutter) để biết API chính xác — mcp_flutter được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
