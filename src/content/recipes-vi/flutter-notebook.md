## Cài đặt Flutter-Notebook

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  Flutter-Notebook: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_notebook/flutter_notebook.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/OpenFlutter/Flutter-Notebook) để biết API chính xác — Flutter-Notebook được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng Flutter-Notebook?

Hãy chọn Flutter-Notebook khi:

- bạn muốn tự động hóa hoặc tối ưu một phần quy trình build
- bạn cần debug, sinh mã hoặc trải nghiệm lập trình tốt hơn
- bạn chuẩn hóa công cụ cho cả nhóm

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `flutter`, `flutter-demo`.
