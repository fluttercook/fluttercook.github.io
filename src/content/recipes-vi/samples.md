## samples là gì?

## Chạy samples

samples là một dự án hoàn chỉnh, bạn có thể clone và chạy. Hãy cài Flutter SDK trước, rồi:

```bash
git clone https://github.com/flutter/samples.git
cd samples
flutter pub get
flutter run
```

Sau đó đọc cấu trúc dự án để xem một app Flutter thực tế được tổ chức ra sao — điều hướng, trạng thái, truy cập dữ liệu và giao diện — rồi fork làm điểm khởi đầu cho app của bạn.
