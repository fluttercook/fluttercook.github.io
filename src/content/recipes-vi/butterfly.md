## Chạy Butterfly

Butterfly là một dự án hoàn chỉnh, bạn có thể clone và chạy. Hãy cài Flutter SDK trước, rồi:

```bash
git clone https://github.com/LinwoodDev/Butterfly.git
cd Butterfly
flutter pub get
flutter run
```
