## agent-device là gì?

CLI to control iOS and Android devices for AI agents. Nó tập trung vào việc đưa AI (trên thiết bị hoặc đám mây) vào ứng dụng Flutter. Dự án nằm tại [callstack/agent-device](https://github.com/callstack/agent-device) và được duy trì bởi `callstack`.

## Cài đặt agent-device

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  agent-device: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:agent_device/agent_device.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/callstack/agent-device) để biết API chính xác — agent-device được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng agent-device?

Hãy chọn agent-device khi:

- bạn thêm chatbot, trợ lý hoặc tính năng sinh nội dung
- bạn cần suy luận trên thiết bị vì quyền riêng tư hoặc offline
- bạn tích hợp LLM hoặc mô hình ML vào ứng dụng

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `adb`, `agentic-ai`, `agents`, `ai-agents`, `android-emulator`, `automation`.
