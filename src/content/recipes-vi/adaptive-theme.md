## Cài đặt adaptive_theme

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  adaptive_theme: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:adaptive_theme/adaptive_theme.dart';
```

Xem thư mục `example/` và [trang pub.dev](https://pub.dev/packages/adaptive_theme) để biết API chính xác — adaptive_theme được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
