## Cài đặt flutter_projects

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  flutter_projects: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_projects/flutter_projects.dart';
```

Xem thư mục `example/` và [trang pub.dev](https://pub.dev/packages/flutter_projects) để biết API chính xác — flutter_projects được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
