## OmniBot là gì?

Your on-phone / mobile AI Agent / Claw, capable of operating terminals and performing a wide range of tasks in the Android world || 你的手机 AI 代理，她可以操作终端，也可以完成 And. Nó tập trung vào việc đưa AI (trên thiết bị hoặc đám mây) vào ứng dụng Flutter. Dự án nằm tại [omnimind-ai/OmniBot](https://github.com/omnimind-ai/OmniBot) và được duy trì bởi `omnimind-ai`.

## Cài đặt OmniBot

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  OmniBot: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:omnibot/omnibot.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/omnimind-ai/OmniBot) để biết API chính xác — OmniBot được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng OmniBot?

Hãy chọn OmniBot khi:

- bạn thêm chatbot, trợ lý hoặc tính năng sinh nội dung
- bạn cần suy luận trên thiết bị vì quyền riêng tư hoặc offline
- bạn tích hợp LLM hoặc mô hình ML vào ứng dụng

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `agent`, `ai`, `alpine`, `android`, `automation`, `bot`.
