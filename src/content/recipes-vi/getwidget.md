## Cài đặt getwidget

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  getwidget: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:getwidget/getwidget.dart';
```

Xem thư mục `example/` và [trang pub.dev](https://pub.dev/packages/getwidget) để biết API chính xác — getwidget được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng getwidget?

Hãy chọn getwidget khi:

- bạn cần một widget dựng sẵn thay vì tự viết từ đầu

- bạn dựng nhanh giao diện ứng dụng di động
