## apidash là gì?

API Dash is a beautiful AI-powered open-source cross-platform (Desktop & Mobile) API Client built using Flutter which can help you easily create & customize you. Nó tập trung vào việc đưa AI (trên thiết bị hoặc đám mây) vào ứng dụng Flutter. Dự án nằm tại [foss42/apidash](https://github.com/foss42/apidash) và được duy trì bởi `foss42`.

## Cài đặt apidash

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  apidash: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:apidash/apidash.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/foss42/apidash) để biết API chính xác — apidash được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng apidash?

Hãy chọn apidash khi:

- bạn thêm chatbot, trợ lý hoặc tính năng sinh nội dung
- bạn cần suy luận trên thiết bị vì quyền riêng tư hoặc offline
- bạn tích hợp LLM hoặc mô hình ML vào ứng dụng

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `api`, `api-client`, `api-testing`, `dart`, `developer-tools`, `flutter`.
