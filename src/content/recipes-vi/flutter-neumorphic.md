## Flutter-Neumorphic là gì?

## Cài đặt Flutter-Neumorphic

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  Flutter-Neumorphic: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_neumorphic/flutter_neumorphic.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/Idean/Flutter-Neumorphic) để biết API chính xác — Flutter-Neumorphic được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng Flutter-Neumorphic?

Hãy chọn Flutter-Neumorphic khi:

- bạn cần một widget dựng sẵn thay vì tự viết từ đầu
- bạn muốn giao diện nhất quán giữa các màn hình
- bạn dựng nhanh giao diện ứng dụng di động

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `android`, `button`, `card`, `concave`, `convex`, `custom-shape`.
