## Cài đặt Airclap

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  Airclap: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:airclap/airclap.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/Gentleflow/Airclap) để biết API chính xác — Airclap được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng Airclap?

Hãy chọn Airclap khi:

- bạn thêm chatbot, trợ lý hoặc tính năng sinh nội dung
