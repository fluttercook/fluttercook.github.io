## Cài đặt flutter-timeline

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  flutter-timeline: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_timeline/flutter_timeline.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/softmarshmallow/flutter-timeline) để biết API chính xác — flutter-timeline được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng flutter-timeline?

Hãy chọn flutter-timeline khi:

- bạn cần một widget dựng sẵn thay vì tự viết từ đầu
- bạn muốn giao diện nhất quán giữa các màn hình
- bạn dựng nhanh giao diện ứng dụng di động

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `calendar`, `customizable`, `events`, `flutter`, `flutter-timeline`, `flutter-widget`.
