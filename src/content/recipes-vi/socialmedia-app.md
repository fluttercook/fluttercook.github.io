## Cài đặt SocialMedia-App

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  SocialMedia-App: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:socialmedia_app/socialmedia_app.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/CharlyKeleb/SocialMedia-App) để biết API chính xác — SocialMedia-App được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng SocialMedia-App?

Hãy chọn SocialMedia-App khi:

- bạn gọi API REST/GraphQL từ ứng dụng Flutter
