## Cài đặt flutter-chat-demo

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  flutter-chat-demo: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_chat_demo/flutter_chat_demo.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/duytq94/flutter-chat-demo) để biết API chính xác — flutter-chat-demo được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng flutter-chat-demo?

Hãy chọn flutter-chat-demo khi:

- bạn gọi API REST/GraphQL từ ứng dụng Flutter
- bạn cần lưu trữ cục bộ, cache hoặc đồng bộ offline
- bạn muốn truy cập dữ liệu có kiểu và dễ kiểm thử
