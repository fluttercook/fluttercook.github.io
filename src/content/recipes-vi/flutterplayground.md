## FlutterPlayground là gì?

Playground app for Flutter. Nó tập trung vào việc thêm chuyển động mượt mà, sinh động cho ứng dụng Flutter. Dự án nằm tại [ibhavikmakwana/FlutterPlayground](https://github.com/ibhavikmakwana/FlutterPlayground) và được duy trì bởi `ibhavikmakwana`.

## Cài đặt FlutterPlayground

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  FlutterPlayground: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutterplayground/flutterplayground.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/ibhavikmakwana/FlutterPlayground) để biết API chính xác — FlutterPlayground được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng FlutterPlayground?

Hãy chọn FlutterPlayground khi:

- bạn muốn micro-interaction đẹp mà không phải tự viết tween
- bạn cần hiệu ứng chuyển trang hoặc loading
- bạn muốn làm sống động giao diện tĩnh

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `animated-icons`, `animatedswitcher`, `animation`, `appbar`, `backdropfilter`, `bottom-navigation`.
