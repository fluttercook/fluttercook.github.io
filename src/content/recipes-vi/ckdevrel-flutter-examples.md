## Cài đặt flutter-examples

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  flutter-examples: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_examples/flutter_examples.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/ckdevrel/flutter-examples) để biết API chính xác — flutter-examples được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng flutter-examples?

Hãy chọn flutter-examples khi:

- bạn cần một widget dựng sẵn thay vì tự viết từ đầu
