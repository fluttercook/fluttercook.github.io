## Cài đặt FlutterUnit

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  FlutterUnit: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutterunit/flutterunit.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/toly1994328/FlutterUnit) để biết API chính xác — FlutterUnit được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng FlutterUnit?

Hãy chọn FlutterUnit khi:

- bạn cần một widget dựng sẵn thay vì tự viết từ đầu
- bạn muốn giao diện nhất quán giữa các màn hình
- bạn dựng nhanh giao diện ứng dụng di động
