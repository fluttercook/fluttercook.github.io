## Cài đặt flutter-ai-rules

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  flutter-ai-rules: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_ai_rules/flutter_ai_rules.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/evanca/flutter-ai-rules) để biết API chính xác — flutter-ai-rules được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng flutter-ai-rules?

Hãy chọn flutter-ai-rules khi:

- bạn thêm chatbot, trợ lý hoặc tính năng sinh nội dung
- bạn cần suy luận trên thiết bị vì quyền riêng tư hoặc offline
- bạn tích hợp LLM hoặc mô hình ML vào ứng dụng
