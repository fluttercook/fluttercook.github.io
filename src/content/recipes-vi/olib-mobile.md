## olib-mobile là gì?

An open-source ebook reader built entirely with AI assistance. Third-party client, frontend interface only. Nó tập trung vào việc đưa AI (trên thiết bị hoặc đám mây) vào ứng dụng Flutter. Dự án nằm tại [shiyi-0x7f/olib-mobile](https://github.com/shiyi-0x7f/olib-mobile) và được duy trì bởi `shiyi-0x7f`.

## Cài đặt olib-mobile

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  olib-mobile: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:olib_mobile/olib_mobile.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/shiyi-0x7f/olib-mobile) để biết API chính xác — olib-mobile được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
