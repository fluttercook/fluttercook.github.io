## Nova3D là gì?

Editable, part-aware 3D generation from text or reference images. Open-source client for nova3d.xyz. Nó tập trung vào việc đưa AI (trên thiết bị hoặc đám mây) vào ứng dụng Flutter. Dự án nằm tại [RareSense/Nova3D](https://github.com/RareSense/Nova3D) và được duy trì bởi `RareSense`.

## Cài đặt Nova3D

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  Nova3D: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:nova3d/nova3d.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/RareSense/Nova3D) để biết API chính xác — Nova3D được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng Nova3D?

Hãy chọn Nova3D khi:

- bạn thêm chatbot, trợ lý hoặc tính năng sinh nội dung
- bạn cần suy luận trên thiết bị vì quyền riêng tư hoặc offline
- bạn tích hợp LLM hoặc mô hình ML vào ứng dụng

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `3d-generation`, `blen`, `cad`, `claude`, `flutter`, `gemini`.
