## appwrite là gì?

## Cài đặt appwrite

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  appwrite: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:appwrite/appwrite.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/appwrite/appwrite) để biết API chính xác — appwrite được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng appwrite?

Hãy chọn appwrite khi:

- bạn thêm chatbot, trợ lý hoặc tính năng sinh nội dung
- bạn cần suy luận trên thiết bị vì quyền riêng tư hoặc offline
- bạn tích hợp LLM hoặc mô hình ML vào ứng dụng

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `android`, `appwrite`, `backend`, `backend-as-a-service`, `docker`, `firebase`.
