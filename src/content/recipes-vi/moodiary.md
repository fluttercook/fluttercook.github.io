## moodiary là gì?

A fully open source cross-platform diary app written by flutter and rust. Nó tập trung vào việc đưa AI (trên thiết bị hoặc đám mây) vào ứng dụng Flutter. Dự án nằm tại [ZhuJHua/moodiary](https://github.com/ZhuJHua/moodiary) và được duy trì bởi `ZhuJHua`.

## Cài đặt moodiary

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  moodiary: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:moodiary/moodiary.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/ZhuJHua/moodiary) để biết API chính xác — moodiary được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng moodiary?

Hãy chọn moodiary khi:

- bạn thêm chatbot, trợ lý hoặc tính năng sinh nội dung
- bạn cần suy luận trên thiết bị vì quyền riêng tư hoặc offline
- bạn tích hợp LLM hoặc mô hình ML vào ứng dụng

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `cross-platform`, `dart`, `diary-app`, `flutter-app`, `rust`, `tensorflow-lite`.
