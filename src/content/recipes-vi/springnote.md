## Cài đặt SpringNote

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  SpringNote: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:springnote/springnote.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/Radiant303/SpringNote) để biết API chính xác — SpringNote được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
