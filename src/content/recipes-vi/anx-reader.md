## Cài đặt anx-reader

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  anx-reader: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:anx_reader/anx_reader.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/Anxcye/anx-reader) để biết API chính xác — anx-reader được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
