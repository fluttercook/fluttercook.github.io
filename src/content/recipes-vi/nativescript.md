## NativeScript là gì?

⚡ Write Native with TypeScript ✨ Best of all worlds (TypeScript, Swift, Objective C, Kotlin, Java, Dart). Use what you love ❤️ Angular, React, Solid, Svelte, Vu. Nó tập trung vào việc cải thiện quy trình phát triển và chất lượng mã Flutter. Dự án nằm tại [NativeScript/NativeScript](https://github.com/NativeScript/NativeScript) và được duy trì bởi `NativeScript`.

## Cài đặt NativeScript

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  NativeScript: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:nativescript/nativescript.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/NativeScript/NativeScript) để biết API chính xác — NativeScript được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng NativeScript?

Hãy chọn NativeScript khi:

- bạn muốn tự động hóa hoặc tối ưu một phần quy trình build
- bạn cần debug, sinh mã hoặc trải nghiệm lập trình tốt hơn
- bạn chuẩn hóa công cụ cho cả nhóm

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `android`, `angular`, `capacitor`, `cross-platform`, `flutter`, `ios`.
