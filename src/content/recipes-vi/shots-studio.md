## Cài đặt shots-studio

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  shots-studio: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:shots_studio/shots_studio.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/AnsahMohammad/shots-studio) để biết API chính xác — shots-studio được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
