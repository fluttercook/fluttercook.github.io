## flutter_html là gì?

A Flutter widget for rendering static html as Flutter widgets (Will render over 80 different html tags!). Nó tập trung vào việc đưa AI (trên thiết bị hoặc đám mây) vào ứng dụng Flutter. Dự án nằm tại [Sub6Resources/flutter_html](https://github.com/Sub6Resources/flutter_html) và được duy trì bởi `Sub6Resources`.

## Cài đặt flutter_html

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  flutter_html: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_html/flutter_html.dart';
```

Xem thư mục `example/` và [trang pub.dev](https://pub.dev/packages/flutter_html) để biết API chính xác — flutter_html được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
