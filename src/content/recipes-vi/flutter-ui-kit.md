## Flutter-UI-Kit là gì?

Flutter app for collection of UI in a UIKit. Nó tập trung vào việc dựng các widget giao diện đẹp, tái sử dụng nhanh hơn. Dự án nằm tại [iampawan/Flutter-UI-Kit](https://github.com/iampawan/Flutter-UI-Kit) và được duy trì bởi `iampawan`.

## Cài đặt Flutter-UI-Kit

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  Flutter-UI-Kit: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_ui_kit/flutter_ui_kit.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/iampawan/Flutter-UI-Kit) để biết API chính xác — Flutter-UI-Kit được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng Flutter-UI-Kit?

Hãy chọn Flutter-UI-Kit khi:

- bạn cần một widget dựng sẵn thay vì tự viết từ đầu
- bạn muốn giao diện nhất quán giữa các màn hình
- bạn dựng nhanh giao diện ứng dụng di động

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `dart`, `flutter`, `uikit`.
