## iconoir là gì?

An open source icons library with 1600+ icons, supporting React, React Native, Flutter, Vue, Figma, and Framer. Nó tập trung vào việc cải thiện quy trình phát triển và chất lượng mã Flutter. Dự án nằm tại [iconoir-icons/iconoir](https://github.com/iconoir-icons/iconoir) và được duy trì bởi `iconoir-icons`.

## Cài đặt iconoir

Thêm gói vào `pubspec.yaml`:

```yaml

```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
