## appflowy-editor là gì?

## Cài đặt appflowy-editor

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  appflowy-editor: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:appflowy_editor/appflowy_editor.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/AppFlowy-IO/appflowy-editor) để biết API chính xác — appflowy-editor được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
