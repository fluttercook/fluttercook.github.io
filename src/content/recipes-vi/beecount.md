## Cài đặt BeeCount

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  BeeCount: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:beecount/beecount.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/TNT-Likely/BeeCount) để biết API chính xác — BeeCount được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
