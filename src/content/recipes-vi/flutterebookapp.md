## Cài đặt FlutterEbookApp

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  FlutterEbookApp: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```

Xem thư mục `example/` và [repo GitHub](https://github.com/JideGuru/FlutterEbookApp) để biết API chính xác — FlutterEbookApp được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
