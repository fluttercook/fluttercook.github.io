## FlutterScreens là gì?

A collection of Screens and attractive UIs built with Flutter ready to be used in your applications. No external libraries are used. Just download, add to your. Nó tập trung vào việc dựng các widget giao diện đẹp, tái sử dụng nhanh hơn. ## Cài đặt FlutterScreens

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  FlutterScreens: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutterscreens/flutterscreens.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/samarthagarwal/FlutterScreens) để biết API chính xác — FlutterScreens được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng FlutterScreens?

Hãy chọn FlutterScreens khi:

- bạn cần một widget dựng sẵn thay vì tự viết từ đầu
- bạn muốn giao diện nhất quán giữa các màn hình
- bạn dựng nhanh giao diện ứng dụng di động

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `dart`, `dartlang`, `flutter`, `flutter-apps`, `flutter-widget`.
