## Cài đặt my_flutter_challenges

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  my_flutter_challenges: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:my_flutter_challenges/my_flutter_challenges.dart';
```

Xem thư mục `example/` và [trang pub.dev](https://pub.dev/packages/my_flutter_challenges) để biết API chính xác — my_flutter_challenges được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
