## Flutter_Pinput là gì?

Flutter package to create Pin code input text field with every pixel customization possibility  with beautiful animations, iOS autofill, Android autofill. Nó tập trung vào việc thêm chuyển động mượt mà, sinh động cho ứng dụng Flutter. Dự án nằm tại [Tkko/Flutter_Pinput](https://github.com/Tkko/Flutter_Pinput) và được duy trì bởi `Tkko`.

## Cài đặt Flutter_Pinput

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  Flutter_Pinput: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_pinput/flutter_pinput.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/Tkko/Flutter_Pinput) để biết API chính xác — Flutter_Pinput được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
