## Prism là gì?

Prism is an open-source Flutter wallpapers and home screen setups app for Android and iOS, with community uploads, curated collections, and AI wallpaper generat. Nó tập trung vào việc đưa AI (trên thiết bị hoặc đám mây) vào ứng dụng Flutter. Dự án nằm tại [Hash-Studios/Prism](https://github.com/Hash-Studios/Prism) và được duy trì bởi `Hash-Studios`.

## Cài đặt Prism

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  Prism: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:prism/prism.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/Hash-Studios/Prism) để biết API chính xác — Prism được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng Prism?

Hãy chọn Prism khi:

- bạn thêm chatbot, trợ lý hoặc tính năng sinh nội dung
- bạn cần suy luận trên thiết bị vì quyền riêng tư hoặc offline
- bạn tích hợp LLM hoặc mô hình ML vào ứng dụng

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `ai`, `android`, `app`, `dart`, `flutter`, `flutter-apps`.
