## Cài đặt Flutter-learning

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  Flutter-learning: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:flutter_learning/flutter_learning.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/AweiLoveAndroid/Flutter-learning) để biết API chính xác — Flutter-learning được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng Flutter-learning?

Hãy chọn Flutter-learning khi:

- bạn cần một widget dựng sẵn thay vì tự viết từ đầu
- bạn muốn giao diện nhất quán giữa các màn hình
- bạn dựng nhanh giao diện ứng dụng di động

Đặc biệt phù hợp nếu dự án của bạn liên quan tới `android-flutter`, `android-ios`, `dart`, `flutter`, `flutter-app`, `flutter-apps`.
