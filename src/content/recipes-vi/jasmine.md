## Cài đặt jasmine

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  jasmine: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:jasmine/jasmine.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/ComicSparks/jasmine) để biết API chính xác — jasmine được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.
