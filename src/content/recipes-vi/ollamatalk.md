## OllamaTalk là gì?

A fully local, cross-platform AI chat application powered by Ollama. Chat securely across macOS, Windows, Linux, Android, and iOS - no cloud required. Nó tập trung vào việc đưa AI (trên thiết bị hoặc đám mây) vào ứng dụng Flutter. Dự án nằm tại [shinhyo/OllamaTalk](https://github.com/shinhyo/OllamaTalk) và được duy trì bởi `shinhyo`.

## Cài đặt OllamaTalk

Thêm gói vào `pubspec.yaml`:

```yaml
dependencies:
  OllamaTalk: ^latest
```

Rồi tải về và import trong mã Dart:

```bash
flutter pub get
```
```dart
import 'package:ollamatalk/ollamatalk.dart';
```

Xem thư mục `example/` và [repo GitHub](https://github.com/shinhyo/OllamaTalk) để biết API chính xác — OllamaTalk được đánh phiên bản kèm tài liệu đầy đủ nên bạn luôn tích hợp đúng bản phát hành hiện tại.

## Khi nào nên dùng OllamaTalk?

Hãy chọn OllamaTalk khi:

- bạn thêm chatbot, trợ lý hoặc tính năng sinh nội dung
