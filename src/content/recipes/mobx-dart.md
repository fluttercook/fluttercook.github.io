## What is mobx.dart?

MobX for the Dart language. Hassle-free, reactive state-management for your Dart and Flutter apps. It focuses on keeping your UI in sync with your app state as it changes. The project lives at [mobxjs/mobx.dart](https://github.com/mobxjs/mobx.dart) and is maintained by `mobxjs`.

## Installing mobx.dart

Add the package to your `pubspec.yaml`:

```yaml

  mobx.dart: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart

```

Check the package's `example/` directory and its [GitHub repo](https://github.com/mobxjs/mobx.dart) for the exact API — mobx.dart is versioned there with full docs so you always integrate against the current release.

## When should you use mobx.dart?

Reach for mobx.dart when:

- your widget tree needs to react to shared, changing data
- you want to separate business logic from presentation
- you're scaling past `setState` and need predictable rebuilds
