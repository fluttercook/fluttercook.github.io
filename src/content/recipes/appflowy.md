## Installing AppFlowy

Add the package to your `pubspec.yaml`:

```yaml

  AppFlowy: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:appflowy/appflowy.dart';
```

Check the package's `example/` directory and its [GitHub repo](https://github.com/AppFlowy-IO/AppFlowy) for the exact API — AppFlowy is versioned there with full docs so you always integrate against the current release.
