API Dash is a beautiful AI-powered open-source cross-platform (Desktop & Mobile) API Client built using Flutter which can help you easily create & customize you. It focuses on bringing on-device or cloud AI features into a Flutter app. The project lives at [foss42/apidash](https://github.com/foss42/apidash) and is maintained by `foss42`.

