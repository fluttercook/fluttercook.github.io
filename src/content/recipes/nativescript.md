## Installing NativeScript

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  NativeScript: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:nativescript/nativescript.dart';
```

Check the package's `example/` directory and its [GitHub repo](https://github.com/NativeScript/NativeScript) for the exact API — NativeScript is versioned there with full docs so you always integrate against the current release.
