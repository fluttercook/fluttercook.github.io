## Installing fvm

Add the package to your `pubspec.yaml`:

```yaml

```

Then fetch it and import it in your Dart code:

```bash

```

Check the package's `example/` directory and its [GitHub repo](https://github.com/conceptadev/fvm) for the exact API — fvm is versioned there with full docs so you always integrate against the current release.
