## What is Nova3D?

Editable, part-aware 3D generation from text or reference images. Open-source client for nova3d.xyz. It focuses on bringing on-device or cloud AI features into a Flutter app. The project lives at [RareSense/Nova3D](https://github.com/RareSense/Nova3D) and is maintained by `RareSense`.

## Installing Nova3D

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  Nova3D: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:nova3d/nova3d.dart';
```

Check the package's `example/` directory and its [GitHub repo](https://github.com/RareSense/Nova3D) for the exact API — Nova3D is versioned there with full docs so you always integrate against the current release.
