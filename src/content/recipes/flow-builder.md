## What is flow_builder?

Flutter Flows made easy! A Flutter package which simplifies navigation flows with a flexible, declarative API. It focuses on structuring screens, routes, and deep links. The project lives at [felangel/flow_builder](https://github.com/felangel/flow_builder) and is maintained by `felangel`.

## Installing flow_builder

Add the package to your `pubspec.yaml`:

```yaml

```

Then fetch it and import it in your Dart code:

```bash

```

## When should you use flow_builder?

Reach for flow_builder when:
