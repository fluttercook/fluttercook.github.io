## What is openai?

Dart/Flutter SDK for ChatGPT and all OpenAI APIs (GPT, Dall-e..). It focuses on bringing on-device or cloud AI features into a Flutter app. The project lives at [anasfik/openai](https://github.com/anasfik/openai) and is maintained by `anasfik`.

## Installing openai

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  openai: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:openai/openai.dart';
```

Check the package's `example/` directory and its [pub.dev page](https://pub.dev/packages/openai) for the exact API — openai is versioned there with full docs so you always integrate against the current release.
