## What is gopeed?

A fast, modern download manager for HTTP, BitTorrent, Magnet, and ed2k. Cross-platform, built with Golang and Flutter. It focuses on talking to APIs, databases, and persistence layers. The project lives at [GopeedLab/gopeed](https://github.com/GopeedLab/gopeed) and is maintained by `GopeedLab`.

## Installing gopeed

Add the package to your `pubspec.yaml`:

```yaml

  gopeed: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:gopeed/gopeed.dart';
```

Check the package's `example/` directory and its [GitHub repo](https://github.com/GopeedLab/gopeed) for the exact API — gopeed is versioned there with full docs so you always integrate against the current release.
