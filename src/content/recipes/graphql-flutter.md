## What is graphql-flutter?

A GraphQL client for Flutter, bringing all the features from a modern GraphQL client to one easy to use package. It focuses on talking to APIs, databases, and persistence layers. The project lives at [zino-hofmann/graphql-flutter](https://github.com/zino-hofmann/graphql-flutter) and is maintained by `zino-hofmann`.

## Installing graphql-flutter

Add the package to your `pubspec.yaml`:

```yaml

  graphql-flutter: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart

```

Check the package's `example/` directory and its [GitHub repo](https://github.com/zino-hofmann/graphql-flutter) for the exact API — graphql-flutter is versioned there with full docs so you always integrate against the current release.

## When should you use graphql-flutter?

Reach for graphql-flutter when:
