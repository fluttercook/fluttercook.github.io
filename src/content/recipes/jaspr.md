## What is jaspr?

Modern web framework for building websites in Dart. Supports SPAs, SSR and SSG. It focuses on building polished, reusable interface widgets faster. The project lives at [schultek/jaspr](https://github.com/schultek/jaspr) and is maintained by `schultek`.

## Installing jaspr

Add the package to your `pubspec.yaml`:

```yaml

  jaspr: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart

```

Check the package's `example/` directory and its [pub.dev page](https://pub.dev/packages/jaspr) for the exact API — jaspr is versioned there with full docs so you always integrate against the current release.
