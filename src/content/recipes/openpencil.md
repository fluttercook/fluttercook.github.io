## Installing openpencil

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  openpencil: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:openpencil/openpencil.dart';
```

Check the package's `example/` directory and its [GitHub repo](https://github.com/ZSeven-W/openpencil) for the exact API — openpencil is versioned there with full docs so you always integrate against the current release.
