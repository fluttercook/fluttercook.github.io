## What is drift?

Drift is an easy to use, reactive, typesafe persistence library for Dart & Flutter. It focuses on talking to APIs, databases, and persistence layers. The project lives at [simolus3/drift](https://github.com/simolus3/drift) and is maintained by `simolus3`.

## Installing drift

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  drift: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:drift/drift.dart';
```

Check the package's `example/` directory and its [pub.dev page](https://pub.dev/packages/drift) for the exact API — drift is versioned there with full docs so you always integrate against the current release.

## When should you use drift?

Reach for drift when:

- you're calling REST/GraphQL APIs from a Flutter app
- you need local storage, caching, or offline sync
- you want typed, testable data access

It's especially relevant if your project touches `dart`, `dart-build-system`, `flutter`, `persistence`, `reactive`, `sqlite`.
