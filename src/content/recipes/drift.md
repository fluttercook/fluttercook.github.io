Drift is an easy to use, reactive, typesafe persistence library for Dart & Flutter. It lives at [simolus3/drift](https://github.com/simolus3/drift) and is maintained by the `simolus3` project.

## Add it to your project

Add the package to your `pubspec.yaml`:


Then run `flutter pub get`. Full API docs and versions are on [pub.dev](https://pub.dev/packages/drift).
