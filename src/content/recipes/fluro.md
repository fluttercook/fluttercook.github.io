## What is fluro?

Fluro is a Flutter routing library that adds flexible routing options like wildcards, named parameters and clear route definitions. It focuses on structuring screens, routes, and deep links. The project lives at [lukepighetti/fluro](https://github.com/lukepighetti/fluro) and is maintained by `lukepighetti`.

## When should you use fluro?

Reach for fluro when:

- you need declarative, URL-driven routing

- you want type-safe routes
