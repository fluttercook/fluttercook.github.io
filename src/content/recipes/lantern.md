## Installing lantern

Add the package to your `pubspec.yaml`:

```yaml

```

Then fetch it and import it in your Dart code:

```bash

```

Check the package's `example/` directory and its [GitHub repo](https://github.com/getlantern/lantern) for the exact API — lantern is versioned there with full docs so you always integrate against the current release.

## When should you use lantern?

Reach for lantern when:

- you need declarative, URL-driven routing
- you're handling deep links or nested navigation
- you want type-safe routes
