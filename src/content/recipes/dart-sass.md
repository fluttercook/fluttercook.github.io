## Installing dart-sass

Add the package to your `pubspec.yaml`:

```yaml

  dart-sass: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart

```

Check the package's `example/` directory and its [GitHub repo](https://github.com/sass/dart-sass) for the exact API — dart-sass is versioned there with full docs so you always integrate against the current release.
