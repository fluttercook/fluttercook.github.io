Modular and customizable Material Design UI components for Flutter. It focuses on building polished, reusable interface widgets faster. The project lives at [material-components/material-components-flutter](https://github.com/material-components/material-components-flutter) and is maintained by `material-components`.

## Installing material-components-flutter

Add the package to your `pubspec.yaml`:

```yaml

  material-components-flutter: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```

Check the package's `example/` directory and its [GitHub repo](https://github.com/material-components/material-components-flutter) for the exact API — material-components-flutter is versioned there with full docs so you always integrate against the current release.
