## What is flutter-permission-handler?

Permission plugin for Flutter. This plugin provides a cross-platform (iOS, Android) API to request and check permissions. It focuses on talking to APIs, databases, and persistence layers. The project lives at [Baseflow/flutter-permission-handler](https://github.com/Baseflow/flutter-permission-handler) and is maintained by `Baseflow`.
