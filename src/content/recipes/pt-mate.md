## What pt_mate is

基于 Flutter（Material Design 3）开发的私有种子站点客户端，支持多种PT站点的种子浏览、搜索和下载管理。目前支持M-Team和NexusPHP类型的站点。 It lives at [JustLookAtNow/pt_mate](https://github.com/JustLookAtNow/pt_mate) and is maintained by the `JustLookAtNow` project.

## Add it to your project

Add the package to your `pubspec.yaml`:

```yaml

```

Then run `flutter pub get`. Full API docs and versions are on [pub.dev](https://pub.dev/packages/pt_mate).
