## What flutter_inappwebview is

A Flutter plugin that allows you to add an inline webview, to use a headless webview, and to open an in-app browser window. It lives at [pichillilorenzo/flutter_inappwebview](https://github.com/pichillilorenzo/flutter_inappwebview) and is maintained by the `pichillilorenzo` project.

## Add it to your project

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  flutter_inappwebview: ^latest
```

Then run `flutter pub get`. Full API docs and versions are on [pub.dev](https://pub.dev/packages/flutter_inappwebview).
