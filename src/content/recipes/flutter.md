## Installing flutter

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  flutter: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:flutter/flutter.dart';
```

Check the package's `example/` directory and its [pub.dev page](https://pub.dev/packages/flutter) for the exact API — flutter is versioned there with full docs so you always integrate against the current release.
