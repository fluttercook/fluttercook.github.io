A Flutter UI Kit（一个 Flutter UI组件库），包含 ScaleView, Swiper, PullRefresh, WaterMark, GradientCircularProgressIndicator... It focuses on building polished, reusable interface widgets faster. ## Installing flukit

Add the package to your `pubspec.yaml`:

```yaml

  flukit: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:flukit/flukit.dart';
```

Check the package's `example/` directory and its [GitHub repo](https://github.com/flutterchina/flukit) for the exact API — flukit is versioned there with full docs so you always integrate against the current release.
