## What cubit is

Cubit is a lightweight state management solution. It is a subset of the bloc package that does not rely on events and instead uses methods to emit new states. It lives at [felangel/cubit](https://github.com/felangel/cubit) and is maintained by the `felangel` project.

## Why it's trending

With 597 stars and 25 forks, cubit is well-established and sits in the top of our State management rankings. Momentum like this usually means a healthy community, real production use, and plenty of examples to learn from.

## Add it to your project

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  cubit: ^latest
```

Then run `flutter pub get`. Full API docs and versions are on [pub.dev](https://pub.dev/packages/cubit).
