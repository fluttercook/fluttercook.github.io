## Installing flutter-action

Add the package to your `pubspec.yaml`:

```yaml

  flutter-action: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart

```

Check the package's `example/` directory and its [GitHub repo](https://github.com/subosito/flutter-action) for the exact API — flutter-action is versioned there with full docs so you always integrate against the current release.
