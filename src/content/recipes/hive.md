## What is hive?

Lightweight and blazing fast key-value database written in pure Dart. It focuses on talking to APIs, databases, and persistence layers. The project lives at [isar/hive](https://github.com/isar/hive) and is maintained by `isar`.

## Installing hive

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  hive: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:hive/hive.dart';
```

Check the package's `example/` directory and its [pub.dev page](https://pub.dev/packages/hive) for the exact API — hive is versioned there with full docs so you always integrate against the current release.

## When should you use hive?

Reach for hive when:

- you're calling REST/GraphQL APIs from a Flutter app
- you need local storage, caching, or offline sync
- you want typed, testable data access

It's especially relevant if your project touches `dart`, `database`, `encryption`, `flutter`, `hive`, `key-value`.
