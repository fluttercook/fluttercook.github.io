## Installing ResponsiveFramework

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  ResponsiveFramework: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:responsiveframework/responsiveframework.dart';
```

Check the package's `example/` directory and its [GitHub repo](https://github.com/Codelessly/ResponsiveFramework) for the exact API — ResponsiveFramework is versioned there with full docs so you always integrate against the current release.

## When should you use ResponsiveFramework?

Reach for ResponsiveFramework when:

- you're calling REST/GraphQL APIs from a Flutter app
- you need local storage, caching, or offline sync
- you want typed, testable data access
