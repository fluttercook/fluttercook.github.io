## Installing conduit

Add the package to your `pubspec.yaml`:

```yaml

  conduit: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart

```

Check the package's `example/` directory and its [GitHub repo](https://github.com/cogwheel0/conduit) for the exact API — conduit is versioned there with full docs so you always integrate against the current release.
