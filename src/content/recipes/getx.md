## What getx is

Open screens/snackbars/dialogs/bottomSheets without context, manage states and inject dependencies easily with Get. It lives at [jonataslaw/getx](https://github.com/jonataslaw/getx) and is maintained by the `jonataslaw` project.

## Add it to your project

Add the package to your `pubspec.yaml`:

```yaml

  getx: ^latest
```

Then run `flutter pub get`. Full API docs and versions are on [pub.dev](https://pub.dev/packages/getx).
