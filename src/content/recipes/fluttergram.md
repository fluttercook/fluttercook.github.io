## Installing fluttergram

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  fluttergram: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:fluttergram/fluttergram.dart';
```

Check the package's `example/` directory and its [GitHub repo](https://github.com/mdanics/fluttergram) for the exact API — fluttergram is versioned there with full docs so you always integrate against the current release.

## When should you use fluttergram?

Reach for fluttergram when:

- you're calling REST/GraphQL APIs from a Flutter app
