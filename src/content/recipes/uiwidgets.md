## What is UIWidgets?

UIWidget is a Unity Package which helps developers to create, debug and deploy efficient, cross-platform Apps. It focuses on building polished, reusable interface widgets faster. The project lives at [UnityTech/UIWidgets](https://github.com/UnityTech/UIWidgets) and is maintained by `UnityTech`.

## Installing UIWidgets

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  UIWidgets: ^latest
```

Then fetch it and import it in your Dart code:

```bash
flutter pub get
```
```dart
import 'package:uiwidgets/uiwidgets.dart';
```

Check the package's `example/` directory and its [GitHub repo](https://github.com/UnityTech/UIWidgets) for the exact API — UIWidgets is versioned there with full docs so you always integrate against the current release.
